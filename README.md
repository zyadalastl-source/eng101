# ENG101.com — الجزري (MEU AI Student Guide)

> Next.js + TailwindCSS project scaffold (Front-End + placeholder Back-End integration)

## Requirements
- Node.js 18+ (recommended)

## Install
```bash
npm install
```

## Run (Development)
```bash
npm run dev
```

Then open: http://localhost:3000

## Notes
- RTL enabled globally (`<html dir="rtl">`).
- MEU brand colors are enforced via Tailwind theme.
- Firebase integration is stubbed in `lib/firebase.ts`.
- Chatbot integration is stubbed in `lib/chatbot.ts`.
- All content is placeholders ("—") by design.
